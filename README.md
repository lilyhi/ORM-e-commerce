# E-commerce Back End

## Purpose
Generate a back end for an e-commerce site using Express.js, Sequelize, and a MySQL database. Testing conducted through Insomnia.

## Built With
* JavaScript
* Node
* Express.js
* Sequelize
* MySQL
* dotenv
